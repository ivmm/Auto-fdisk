# Auto fdisk
Auto fdisk for SpacePack Tools

## Demo

![Auto fdisk](http://i2.muimg.com/567571/d08804b00c13bfa5.png) 

## How to use

```bash
$ wget https://raw.githubusercontent.com/Vtrois/Auto-fdisk/master/auto_fdisk.sh
$ chmod +x auto_fdisk.sh
$ ./auto_fdisk.sh
```
